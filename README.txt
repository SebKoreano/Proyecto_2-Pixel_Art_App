# Proyecto 2: Pixel Art App

Este es un proyecto de una aplicación de arte en píxeles desarrollada en Python. La aplicación permite a los usuarios crear y editar imágenes pixeladas de manera sencilla e intuitiva.

## Características

- **Diseño de lienzo personalizado:** Los usuarios pueden definir el tamaño del lienzo pixelado.
- **Selección de colores:** Paleta de colores para elegir y aplicar a los píxeles.
- **Herramientas de dibujo:** Incluye herramientas básicas como lápiz, borrador y relleno.
- **Guardar y cargar proyectos:** Los usuarios pueden guardar sus creaciones y cargarlas posteriormente para continuar editándolas.
- **Exportar imágenes:** Exporta las imágenes creadas en formatos populares como PNG.

## Uso

- Abre la aplicación y define el tamaño del lienzo.
- Usa las herramientas de dibujo y la paleta de colores para crear tu arte en píxeles.
- Guarda tu progreso y exporta tus imágenes según sea necesario.
